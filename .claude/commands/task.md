# Task

Создай новую задачу в формате markdown.

## Аргументы
$ARGUMENTS — название или описание задачи

## Действия

1. **Создай файл задачи** в `docs/tasks/`:
   - Имя файла: `YYYY-MM-DD-краткое-название.md`
   - Используй текущую дату

2. **Заполни шаблон**:
   ```markdown
   # [Название задачи]

   ## Описание
   [Подробное описание что нужно сделать]

   ## Критерии готовности
   - [ ] Подзадача 1
   - [ ] Подзадача 2
   - [ ] Тесты написаны и проходят
   - [ ] Код отревьюен

   ## Заметки
   [Дополнительная информация, ссылки, идеи]

   ## Статус
   - Создано: [дата]
   - Статус: В работе / Готово / Отложено
   ```

3. **Если директория не существует**, создай её:
   ```bash
   mkdir -p docs/tasks
   ```

4. **Покажи путь** к созданному файлу
