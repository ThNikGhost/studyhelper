# Session Start

Начни новую сессию работы над проектом.

## Действия

1. **Прочитай файлы состояния:**
   - `docs/PROJECT_STATUS.md` — общий статус проекта
   - `docs/CURRENT_TASK.md` — текущая задача (если есть)
   - `docs/DECISIONS.md` — принятые решения

2. **Выведи краткое резюме:**
   - Текущее состояние проекта (1-2 предложения)
   - Что было сделано в прошлой сессии
   - Какая задача сейчас в работе (если есть)

3. **Спроси:**
   - "Продолжаем работу над [текущая задача]?"
   - Или "Над чем работаем сегодня?"

## Формат ответа

```
**Статус проекта:** [краткое описание]

**Последняя сессия:** [что было сделано]

**Текущая задача:** [название] — [прогресс %]

Продолжаем работу над этой задачей или переключаемся на другое?
```
