# Планы на будущее — StudyHelper

## Фаза 2: Расширенный функционал

### Push-уведомления
**Приоритет:** Высокий

- Уведомления о дедлайнах:
  - За 3 дня до дедлайна
  - За 1 день до дедлайна
  - В день дедлайна
- Уведомления об изменениях расписания
- Утренняя сводка (настраиваемое время, по умолчанию 8:00)
- Настройки уведомлений для каждого пользователя

**Технологии:** Web Push API, pywebpush, Celery Beat

---

### Файловое хранилище
**Приоритет:** Высокий

- Загрузка файлов до 50 MB
- Категории: методичка, задачник, лекции, шпаргалки, прочее
- Привязка к предметам (опционально)
- Массовая загрузка (drag & drop)
- Просмотр PDF, изображений в браузере
- Скачивание файлов
- Поиск по названию

**Технологии:** FastAPI UploadFile, локальное хранилище или S3

---

### Посещаемость
**Приоритет:** Средний

- Автоматическая отметка "посетил" после окончания пары
- Ручная отметка пропуска (кнопка "Я пропустил")
- Статистика: посетил X из Y пар
- Статистика по каждому предмету
- Визуализация (прогресс-бар, график)

---

### Календарь
**Приоритет:** Средний

- Агрегация данных: расписание + дедлайны работ
- Виды: день, неделя, месяц
- Цветовая кодировка по типам событий
- Быстрый переход к работе/паре

---

### Прогресс и визуализация
**Приоритет:** Средний

- Прогресс по каждому предмету (% выполненных работ)
- Общий прогресс семестра
- Сравнение прогресса с партнёром
- Графики и диаграммы

---

### Учебный план (семестры)
**Приоритет:** Низкий

- CRUD для семестров
- Переключение текущего семестра
- Автоматическое определение текущего семестра по датам
- Архивирование старых семестров

---

## Фаза 4: Масштабирование

### Виджеты для телефона
**Приоритет:** Средний

**iOS (WidgetKit):**
- Требует нативного приложения или PWA + Shortcuts

**Android (App Widgets):**
- Можно реализовать через TWA (Trusted Web Activity)

**Содержимое виджетов:**
- Следующая пара с таймером
- Ближайший дедлайн
- Быстрые действия

---

### Telegram-бот
**Приоритет:** Высокий

**Функции:**
- Уведомления (дублирование push)
- Команды:
  - `/today` — расписание на сегодня
  - `/next` — следующая пара
  - `/deadlines` — ближайшие дедлайны
  - `/status` — обновить статус работы
- Inline-кнопки для быстрых действий

**Технологии:** python-telegram-bot, aiogram

---

### AI-агент для голосовых команд
**Приоритет:** Низкий

**Возможности:**
- "Какая следующая пара?"
- "Добавь работу по матану на завтра"
- "Отметь лабу как сделанную"
- Умные напоминания на основе поведения

**Технологии:** OpenAI API / Claude API, Whisper для STT

---

### Автоматический расчёт автомата
**Приоритет:** Низкий

- Парсинг требований к автомату (если структурированы)
- Подсчёт текущего прогресса
- Рекомендации: "Осталось сдать 2 лабы для автомата"

---

### Интеграция с Google Calendar
**Приоритет:** Средний

- Экспорт расписания в Google Calendar
- Экспорт дедлайнов как событий
- Синхронизация (опционально)
- Альтернатива: экспорт в iCal формат

---

### Расширение на других пользователей
**Приоритет:** Низкий (далёкое будущее)

**Варианты:**
1. Многопользовательский режим с группами
2. Публичный сервис для студентов ОмГУ
3. Шаблон для других университетов

**Требования:**
- Доменное имя
- SSL сертификат
- Масштабируемая архитектура
- Модерация контента
